CSS Table

--

HTML Table can be improved with CSS

-
1 ) To specify table border we use border property.
2 ) To use full-width table we give width:100% ;
3 ) Border-collapse : collapse ; is used to collapse the border
4 ) Width and height property is used to set the height and width of the table and value can be in pixel or percentage.
5 ) By default, the content of <th> is center-aligned and the content of <td> is left-aligned
6 ) text-align : left/right/center; 7 ) By default, in vertical (top,bottom,middle) alignment , th and td have middle alignment. 8 ) To control the space between content and border , we use padding property on <td> and <th> elements.

tr:hover{background-color:red;} ---- Hoverable table


For zebra stripped table use nth-child() selector and add background color
tr:nth-child(even){background-colr:red;}


Responsive Table --


A responsive table will display a horizontal scroll bar if the screen is too small to display the full content
add a container element(like div) with workflow-x:auto; around the table element to make it responsive
