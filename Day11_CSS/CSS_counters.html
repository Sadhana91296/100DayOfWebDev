CSS Counters

CSS counters are like variable. The variable value can be incremented by rules.(Which will track how many times they are used.)
  • Counter-reset : Creates or reset the counter
  • Counter-increment : increment a counter value
  • Content : Insert generated content
  • Counter() or Counters() function : Adds the value of counter to an element
To use CSS counter, it must first be created with counter-reset
