An HSL color value is specified with the hsl() function, which has the following syntax: hsl(hue, saturation, lightness)

Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green, 240 is blue. Saturation is a percentage value; 0% means a shade of gray and 100% is the full color. Lightness is also a percentage; 0% is black, 100% is white.

HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of the object. An HSLA color value is specified with the hsla() function, which has the following syntax: hsla(hue, saturation, lightness, alpha) The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
