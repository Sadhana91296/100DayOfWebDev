An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out".


CSS has the following outline properties -
outline-style : Define the style of the outline -
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset
  • none
  • hidden

outline-width :This property defines the width of the outline
  • thin(1px)
  • medium(3px)
  • Thick(5px)
  • A specific size(px,em,cm etc.)

outline-color : Used to set the color of the outline.
  • name : Specify the color name, like "red"
  • HEX : Specify the color value in hex code, like #FF0000
  • rgb : Specify color value with rgb() method, like rgb(255,0,0)
  • HSL : Specify HSL value with hsl() method, like hsl(0,100%,50%)
  • invert : Perform a color inversion, which ensure that outline is visible, regardless of color background

outline-offset : This property adds space between outline and edge/Border of an element.This space is transparent

outline : outline property is shorthand property for setting width,style color in one property.
outline : 5px solid black;
