The CSS height and width property is used to set the height and width of elements.
CSS max-width property is used to set the maximum width of elements.
Height and width properties does not include the border, padding and margin properties,
Height and width have following value : -
  • auto - This is default,browser swt the height and width automatically
  • length - Define the height and width in px or cm etc.
  • % - define height and width in percent of the containing block
  • initial - set the height and width default value
  • inherit - height and width is inherited from its parent value

max-width handle the small window problem


same way we can also set the min-height or min-width of elements block.
