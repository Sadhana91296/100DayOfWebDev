Step 1- First of all, you must find the element.This is done by using the HTML DOM method getElementById();

Step 2- We need a drawing object for the canvas, The getContext() is a built-in HTML object, with properties and methods for drawing.
var ctx=canvas.getContext("2d");

Step 3- Now you can on the canvas, see the fill style of the drawing object to the color red:
ctx.fillStyle="#FF0000";

The fillStyle property can be a css color, a gradient, or a pattern. The default fillStyle is black.

The fillRect(x,y,width,height) method draws a rectangle, filled with the fill style, on the canvas.
