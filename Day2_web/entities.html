Reserved character in HTML must be replaced with HTML entities. Some character are reserved in HTML.If you use greater then or less than sign then the browser might get confused with tag symbol. So in HTMl for reserved key we used the elements entities. An entity look like-

&#entity_number;

or &entity_name;

non-breaking space -  
greater than symbol - >
lesser than symbol - <
ampersand symbol - &
double quote symbol - "e;
single quote symbol - '

registered trademark symbol - ®
