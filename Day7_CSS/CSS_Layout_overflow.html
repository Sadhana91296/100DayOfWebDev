The CSS overflow property control what happens to content that is too big to fit into an area
The overflow property specify wheather to clip the content or add scrollbar, when the content of an
element is too big to fit in specified area.

Overflow : visible

--

By default the overflow is visible, meaning that it is not cliped and it's render outside the box.


You can use the overflow property when you want to have better control of the layout.
