Position property specifies the positioning method used for an element.(static,relative,fixed,absolute,sticky)

Position : Static --

HTML element are positioned static by default,Static element are not affected by top,bottom,right , left properties.
Static position is not positioned in any special way, It is always positioned according to the normal flow of the page.

Position : Fixed --

An element with positioned fixed is positioned with realtive viewport, Which means it always stay in same place even if the page scrolled.
The top, right, bottom, left properties are used to positioned the element.
The fixed element does not leave the gap, where it would normally have been loacted.
This is fixed division.

Position : relative --

The element with position:relative is positioned relative to its normal position.
Setting the top, right, bottom, left properties cause the relatively positioned element position to its normal position.

Position : sticky --

An element with position sticky, is positioned based on the user scroll position .
A sticky element toggles between relative and fixed, depending on the scroll position.
It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).

Position : absolute --

An element with position absolute, is positioned relative to nearest positioned ancestor.However, if absolute element have no any ancestor
it uses the document body and moves along with page scrolling.
